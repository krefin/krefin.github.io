About

Saya adalah merupakan seorang pemuda desa yang kelahiran tahun 2002 dan berasal dari Cibingbin, Kab. Kuningan Jawa barat lalu sedang menempuh pendidikan S1 di Universitas Kuningan angkatan tahun 2021. Saya memilih untuk masuk ke Fakultas Ilmu Komputer dan memilih program studi Sistem Informasi karena ingin mempelajari materi tentang web developer dan bisnis agar semakin menguasai lagi dibidang tersebut. Sehingga kelak saya akan menjadi seorang Web depeloper.

Saya mempunyai hobi olahraga dan photography. Untuk olahraga saya suka sekali bermain basket, karena dulu saat SMP saya mengikuti ekskul basket. Sedangkan untuk photography mulai SMK saya mengikuti ekskul multimedia dan saya sangat menyukai bidang photography, dan videography. Namun diantara photography dan photography, photographylah bidang yang paling saya sukai. Dan sampai saat ini menjadi salah satu hobi saya.
